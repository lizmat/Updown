NAME
====

Updown - provide basic API to Updown.io

SYNOPSIS
========

```raku
use Updown;

my $ud = Updown.new;
```

DESCRIPTION
===========

Updown provides a simple object-oriented interface to the API as provided by [Updown.io](Updown.io).
